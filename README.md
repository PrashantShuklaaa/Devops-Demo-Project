# 🚀 CI/CD Pipeline with Docker

## 📌 Project Overview
This project demonstrates a CI/CD pipeline using GitHub Actions and Docker.

## ⚙️ Tech Stack
- Node.js
- Docker
- GitHub Actions
- Docker Hub

## 🔁 CI/CD Workflow
1. Code pushed to GitHub
2. GitHub Actions triggered
3. Docker image built automatically
4. Image pushed to Docker Hub

## 🐳 Docker Commands (Manual)
docker build -t devops-app .
docker run -p 3000:3000 devops-app

## 🤖 Automation
- Automated build and push using GitHub Actions
- No manual Docker commands required after setup

## 🌍 Docker Hub Image
https://hub.docker.com/r/prashuk370/devops-app

## 📈 Outcome
- Reduced manual effort
- Fully automated build and deployment pipeline
